Settings Network
this settings page contains various network related settings

Proxy server
here you may specify a http proxy server to be used for network requests if desired (no "http://" prefix, just the plain server name)

Proxy port
here you may specify the port the proxy server is listening to

Proxy user name
here you may set the user name for autentification to the proxy server (if required)

Proxy password
here ou may set the password for autentification to the proxy server (if required). NOTE: the password is never stored/restored. you have to reenter it in any new TableCrab session

Fetch timeout
here you may adjust the time TableCrab tries to fetch a network request before giving up (in seconds)
