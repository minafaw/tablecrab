Hand Viewer
This tab is a bit special. behind the sceeneds there is a grabber running that will check if instant hand history is open. if so, the currently selected hand is dumped nicely formatted to this tab. best use along with hotkey Instant Hand History as a replacement for PokerStars replayer. the hand viewer may be adjusted via settings hand viewer and settings hand viewer style sheet.
  • the viewer only works for Texas Hold'em and Omaha (nl/pl)
  • make shure "Show Summary Only" is unchecked in PokerStars instant hand hitory
