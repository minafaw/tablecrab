PokerStars ini decrypter
decrypter / encrypter for PokerStars ini files. hit the Decrypt Directory.. button to recursively decrypt all ini files in a directory of your choice. the decrypted ini files will be dumped as IniName-Tc2.ini into that directory (that is postfixed with "-Tc2"). you may encrypt all ini files again by hitting the Encrypt button or you may select a directory to encrypt all ini files by hitting the Encrypt Directory.. button.
  • the decrypter may stop working at any time (stars may change the encryption key for example)
  • the decrypter adds some data as comment to the very begining of the decrypted files. this data is required for reincrypting, so don't change or remove it!
  • the decrypter may may not be able to decrypt files encrypted with other ini decrypters or the other way round.
