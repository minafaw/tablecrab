Hotkey Table Size Next
whenever you hit this key the current table will be resized to the size of the next template on the Setup tab. for example, if you have two or more templates of differing sizes you can use this hotkey to adjust tables sizes on the fly.
  • linux users: this hotkey is not working in wine-1.1.4.2 up to wine-1.3.0. you may have to up- or downgrade wine.
