Settings Global
Backup TableCrab
you may backup TableCrab (all settings, all templates, all hotkeys ...) to a config file of your choice. this config file may be passed to TableCrab later by adding the "--config" option to the commandline. any changes to tableCrab will from then on be stored in the file you passed. 
>> TableCrab.exe --config "c:\Path\To\Your\TableCrab.ini"

Global Style
here you may adjust the overall looks of TableCrab.

Global Font
here you may adjust the font that TableCrab uses.

Fixed Font
here you may adjust the font that is used for fixed font text blocks like the raw hand history in the hand viewer.

Zoom Steps
some widgets of TableCrab allow for zooming in and out. you may adjust the number of steps available here.

Restore Mouse Position
Restores the mouse cursor position whenever TableCrab had to move the mouse around.

Alternating Row Colors
some widgets of TableCrab may show alternating row colors. here you can turn this feature on or off.

Child Item Indicators
some tree widgets of TableCrab show these little [+][-] boxes at the root item. if you don't like this turn it off here (they can still be expanded/collapsed by hitting "+" or "-" on your keyboard).
