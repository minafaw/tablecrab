Flopster
a tool for board card reading

the tool displays boards cards (flop / turn / river). you may deal the next street by hitting Next (left arrow key) or deal random card(s) on the current street by hitting Random (right arrow key). you may enter / alter cards (case sensitive) by filling in the edit box next to the "Next" button and hitting return. or zoom in / out by adjusting the value in the Zoom box, adjust the Felt color, or adjust the card Border color.

if you want to use Custom cards, select a directory containing (png, jpg, svg, bmp) card images. the card images should be named like this: "2h.png" ... "As.png" that is CardRankCard+Suit.imageFormat. if you like to return to the built in cards, remove the custom directory from the edit box.
