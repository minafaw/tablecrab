Hotkey Card Protector
this hotkey will show / hide a window that you can use for example to cover pocket cards in training sessions.
  • note that the window will always be topmost of all other windows
  • you can only have one card protector at a time
  • the hotkey does not work if the main window of TableCrab has the keyboard focus. for lnux users: the hotkey does not work if a linux window or the main window of TableCrab has the keyboard focus.
