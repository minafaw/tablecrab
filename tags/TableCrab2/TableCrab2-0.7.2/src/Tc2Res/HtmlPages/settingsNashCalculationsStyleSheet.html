Settings Nash Calculations Style Sheet
here you may adjust the style sheet for the nash calculations in the Hand viewer tab.

i assume you are familiar the with style sheets and know what you are doing when editing the box. you may Open a style sheet, you may Save the current one to file or Restore the default one.
  • changes will take effekt on the next hand that is loaded
  • the default style sheet may change with every relase. so it is maybe be a good idea to save your changes to file and restore default to see what changed in new releases.
