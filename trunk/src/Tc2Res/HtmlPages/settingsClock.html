Settings Clock
here you can adjust the little clock ticking on the status bar

On
check to set the clock on, uncheck to set it to off

Speed
sets the tick speed of the clock (in seconds)

Precission
sets the precission of the clock (in digits)

Increment
sets the amount to increment the clock on each tick

Random mode
check to set the clock to random mode. that is on each tick it will randomly pick a number according to settings bove.

Foreground color
adjusts the foreground (text) color of the clock. you may reset the color at any time

Background color
adjusts the background color of the clock. you may reset the color at any time

  • the clock is no high precission device - it ticks along with reasonable accuracy
