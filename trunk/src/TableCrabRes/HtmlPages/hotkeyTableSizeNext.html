Hotkey Table Size Next
whenever you hit this key the current table will be resized to the size of the next template on the Setup tab. for example, if you have two templates of different sized tables you can use this hotkey to toggle your tables size on the fly.
  • linux users: this hotkey does nothing in wine-1.4.2 up to wine-1.3.0. you may have to upgrade wine. currently there is no release > 1.3.0 so you have to grab and compile wine directly from the repositiory.
