Settings Hand Css
here you may adjust the style sheet for the Hand tab.

i assume you are familiar the with Css format and know what you are doing when editing the box. you may Open a style sheet, you may Save the current one to file or Restore the default one.
  • changes will take effekt on the next hand that is loaded
i may make adjustements to the default Css on every TableCrab release. so it may be a good idea to save the current Css and hit Restore Default from time to time to see what has changed.
