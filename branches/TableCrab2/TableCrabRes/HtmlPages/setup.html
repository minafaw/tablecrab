Setup
as said, TableCrab does not know much about tables. it does not try to guess where buttons or checkboxes are. it can only work on fixed sized windows and fixed positions. for this you have to set up templates for.

the process is fairly simple. create a new template and sit at a table. hit the screenshot hotkey when buttons become visible and go back to the setup tab. select the button or checkbox in the template you want to assign and double click the corrosponding corrosponding button or checkbox on the screeenshot. this will fill in the location in the template. you may need more than one screenshot. and it's a good idea to save them for later use.
  • you can setup up to 64 templates
  • TableCrab does not check for conflicting templates. it always takes the first one in the list if in doubt.
  • use PokerStars custom layouts to make shure your tables are of a fixed size because..
  • a template will only work for a table of its size
  • buttons or chekboxes with a location of "None" will be ignored
Toolbar
New
creates a new table template. select the template you want to create from the dropdown menu.

Up
moves the curren template up.

Down
moves the current template down.

Remove
removes the current template.

Save Screenshot..
saves the current screenshot to file.

Open Screenshot..
opens a screenshot from file.

Info..
opens a dialog with detailed info about the window the screenshot was taken from.
