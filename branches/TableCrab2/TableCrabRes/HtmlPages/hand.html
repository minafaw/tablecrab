This tab is a bit special. behind the sceeneds there is a grabber running that will check if PokerStars InstantHandHistory dialog is open. if so, the currently selected hand is dumped nicely formatted to this tab. best use along with the InstantHandHistory hotkey as a replacement for PokerStars replayer.
  • the grabber works only for Texas Hold'em and Omaha (nl/pl)
