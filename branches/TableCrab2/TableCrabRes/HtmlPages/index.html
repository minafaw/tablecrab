TableCrab - PokerStars hotkeys and more
Welcome to TableCrab. TableCrab is a simple hotkey manager for PokerStars and some extras.
  • it is developed and tested on linux and thus runs on wine
  • it is tested on windowsXP
  • it comes equipped with a hand viewer to replace PokerStars replayer
  • ...
TableCrab does not know much about PokerStars tables so to make TableCrab work you have to define tables on the setup tab.
Security
TableCrab does not access the internet. point.
