TableCrab - PokerStars hotkeys and more
Welcome to TableCrab. TableCrab is a hotkey manager for PokerStars including some extras.
  • it is developed and tested on linux and thus runs on wine
  • it is tested on windowsXP
  • it comes equipped with a hand viewer to replace PokerStars replayer
  • ...
TableCrab does not know much about PokerStars tables. it works on fixed sized windows and clicks buttons at positions you specify. to do so you have to define one or more templates in the setup tab and use PokerStars custom layouts to keep the tables at a fixed size.

after doing so you can start setting up hotkeys on the hotkeys tab.
Security
TableCrab does not access the internet. never.
