Settings PokerStars
Auto-Close Popup News
if checked, TableCrab will try its best to automatically close news boxes

Auto-Close Tourney Registration Boxes
if checked, TableCrab will try its best to automatically close these OK boxes popping up on tourney registration / unregistration

Auto-Close Table Message Boxes
if checked, TableCrab will try its best to automatically close these OK boxes popping up at tables

Auto-Close Log In Box
if checked, TableCrab will try its best to automatically close these log in box. that is, you hit "Login" in the PokerStars lobby and if you user name and password is filled in in the box popping up TableCrab will automatically hit OK for you.
