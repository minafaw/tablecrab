Hotkeys
to setup a hotkey select "New" from the toolbar and fill in the dialog that pops up. to assign keyboard keys select the "Hotkey" box in the dialog and start typing the desired keys or key combinations on your keyboard.

the keys you entered may look funny. "<LeftControl+Oem-Period>" for example. this is because TableCrab is too lazy to look up the keys as they read on your keyboard. use the "Hotkey Name" box to overwrite with a name of your choice. for the above key "LeftCtrl+Period" could be a nice name.

some special keys can be found in the dropdown of the "Hotkey" box:
  • MouseWheelUp - triggers the action when the mouse wheel is scolled upwards
  • MouseWheelDown - - triggers the action when the mouse wheel is scolled downwards
  • keys that would trigger actions in the dialog when you would try to enter them
you can always go back to entering keys directly from your keyboard by selecting the first item in the dropdown.
  • you can setup up to 64 hotkeys
  • "LeftControl" is "left control key". that is left not right. same goes for all other keys that have left and a right representation on your keyboard.
  • TableCrab does not check for conflicting hotkeys. it always takes the first one in the list if in doubt.
  • you can not assign the following keys: Shift, Scroll, Numlock and Capital. TableCrab ignores these keys and does not know if they are locked on your keyboard or not.
  • hotkeys are disabled as long as the mouse is over the chat or notes box.

Toolbar
New
creates a new hotkey. select the hotkey you want to create from the dropdown menu.

Up
moves the current hotkey up.

Down
moves the curren hotkey down.

Remove
removes the current hotkey.
