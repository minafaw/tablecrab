PokerStars FPP calculator
simple PokerStars FPP calculator to convert FPPs to cash or vice versa for a specified game type

usage:
  1. select VIP level for the conversion
  2. select game type
  3. select game type currency
  4. choose if you want to convert from FPPs to Cash or vice versa
  5. enter amount
  6. Calculate!
the result will be shown in the box next to the calculate button and be cached in the edit box below for better overviews. you may clear the cached results at any time.
