Hand history viewer
simple PokerStars hand history file viewer. you may open a hand history file, save it or save (the current) hand as html. the spin box on the top can be used to navigate through the file.

for sitNgo tournies you may fetch nash calculations for a hand from HoldcemResources.net. to do so either select a payout structure or fill in a custom one. this automatically triggers fetching of stats on every new hand loaded.

if everthing goes well the data for the hand will be displayed in the browser as taken from HoldemResources.net icm calculator. you may stop fetching data at any time by selecting "None" from the combo box.
  • the data is fetched from the internets - this may take some time or may fail at any time. so browsing a hand history file may become considerably slower
  • the hand history browser does not know if the hand history displayed is actually a sitNgo. if you fetch data for a non sitNgo results are undefined.
