Hand history viewer
simple PokerStars hand history file viewer. you may open a hand history file, save it or save (the current) hand as html. the spin box on the top can be used to navigate through the file.

for sitNgo tournies you may fetch nash calculations for a hand from HoldcemResources.net by filling in the credentials on the right pane of the hand viewer.
  1. select the payout structure from the combo box or fill it in by hand in the wedit box to the right
  2. if desired specify in a proxy server to use for fetching data
if everthing goes well the data for the hand will be displayed in the broweser as taken from HoldemResources.net icm calculator. you may stop fetching data at any time by selecting "None" from the combo box.
  • the data is fetched from the internets - this may take some time or may fail at any time. so browsing a hand history file will become considerably slower
  • the hand history browser does not know if the hand history displayed is actually a sitNgo. if you fetch data for a non sitNgo results are undefined.
