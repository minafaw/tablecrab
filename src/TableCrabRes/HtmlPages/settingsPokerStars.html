Settings PokerStars
Auto-Close Popup News
if checked, TableCrab will try its best to automatically close news boxes

Close Tourney Registration Boxes
if checked, TableCrab will try its best to automatically close these OK boxes popping up on tourney registration / unregistration

Close Table Message Boxes
if checked, TableCrab will try its best to automatically close OK boxes popping up at tables

Close Log In Box
if checked, TableCrab will try its best to automatically close these log in box wherever it may pop up. for example: you hit "Login" in the PokerStars lobby and if you user name and password is filled in in the box popping up TableCrab will automatically hit OK for you.

Move Mouse To Active Table
if checked, TableCrab will move the mouse pointer to the point EmptySpace at the active table. WARNING: enabling this may cause all kinds of unwanted side effects. windows may get thrown around and what not. i assume you know what you are doing.
